Início rápido

Para vê os exemplos funcionando, basta inicializar o servidor, na pasta./examples/server, com o comando make build

Exemplo

              
                $ cd ./examples/server
                $ make build

Depois abra o seu navegador no endereço http://localhost:3000

Exemplos principais

Veja abaixo uma lista com os principais exemplos do códio.

Caso queira fazer alterações, em cada pasta há um arquivo Makefile com o comando build, ou seja, basta abrir o terminal na pasta do exemplo e digitar:

              
                $ make build

Os exemplos são:

Exemplo Descrição
Belzié Curve Mostra como usar o algoritmo de Belzié para gerar uma curva e em seguida movimentar uma div com o easing tween, ou, facilitador de interpolação.
highlighting Show syntax highlight on markdown text.
prettyTable Render table with Scribbler’s pretty table style.
